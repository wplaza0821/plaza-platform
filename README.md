# Plaza Construction Platform

A clean, mobile-first construction project management platform for Plaza and Associates.

**Live URL:** https://wplaza0821.github.io/plaza-platform/

## Features
- Projects Dashboard
- Schedule of Values / Payment Applications (AIA G702/G703)
- RFIs (Requests for Information)
- Submittals
- Change Orders
- Permit Plans & Specifications

Built for Plaza and Associates — forensic building enclosure engineering.
Single-file HTML, no dependencies, data stored in browser localStorage.
